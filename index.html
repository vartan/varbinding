VarBinding 0.0.0 A proof of concept

The concept

This is a proof of concept for an event-based two-way-binding. This system allows you to bind a javascript variable to an HTML DOM element, and when either changes value, it will update the other instantly. This system is very simple:

  1. var myBoundVariable = VarBinding()
  2. myBoundVariable.set("This is my value")
  3. var myElement = document.getElementById("myElementID")
  4. myBoundVariable.bind(myElement)
or more succinctly, var myBoundVariable = VarBinding("This is my value").bind(document.getElementById("myElementID"))
